# 📚 PitStop Library System

A modern and responsive **Library Management System** built using **React + TypeScript + Vite** with authentication support and clean UI design.

---

## 🚀 Features

* 🔐 User Authentication
* 📖 Book Management
* 👨‍🎓 Student/User Dashboard
* 🔍 Search & Filter Books
* 📱 Responsive UI
* ⚡ Fast Performance using Vite
* 🎨 Modern Interface

---

## 🛠️ Tech Stack

| Technology | Usage              |
| ---------- | ------------------ |
| React      | Frontend Framework |
| TypeScript | Type Safety        |
| Vite       | Build Tool         |
| CSS        | Styling            |
| Node.js    | Backend Runtime    |

## ⚙️ Installation

```bash
# Clone Repository
git clone https://github.com/your-username/pitstop-library.git

# Open Folder
cd pitstop-library

# Install Dependencies
npm install

# Run Development Server
npm run dev
```

---

## ▶️ Run Backend

```bash
npm run server
```

## 🌐 Environment Variables
